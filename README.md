## Description
My first TryHackMe room. A simple intro to offensive security where I learned how to find hidden pages and think like a hacker.

## Overview
This room introduces the basics of offensive security by simulating a simple web application attack. The goal is to think like an attacker and understand how vulnerabilities are discovered and exploited.

## Objectives
* Learn how attackers think.
* Understand basic web application vulnerabilities.
* Perform simple enumeration and exploitation.

* Small misconfigurations can lead to full access.
* Thinking like a hacker helps in securing systems effectively.
